设计思想

驴友足迹 是一个分享旅游时所见所闻所感为主题的网站。

网站主要包含以下页面：首页index.html、景廊Gallery.html、感悟Testimonials.html、注册reg.html 和 本页面

首页:主要展示了一周最佳美景、旅游达人所见所拍的景色集等

景廊:展示了大量的旅途所拍摄的高清美景

感悟:按条分享了最火热的旅游感悟

注册:用JavaScriot进行注册信息验证

主要采用了HTML、DIV+CSS、JavaScript、Jquery、XML技术
